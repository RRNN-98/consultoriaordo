# ORDO Premium Website

Este é o site institucional premium desenvolvido para a ORDO Consultoria.

## Estrutura do Projeto

- `index.html`: Arquivo principal com a estrutura do site.
- `styles.css`: Folha de estilos com design system dark premium.
- `script.js`: Interações, efeitos 3D e animações.
- `assets/`: Pasta contendo imagens e o logo.

## Tecnologias Utilizadas

- HTML5 Semântico
- CSS3 (Variáveis CSS, Flexbox, Grid, Animações)
- JavaScript (Vanilla) para interatividades leves

## Funcionalidades

- Design responsivo (Desktop e Mobile)
- Efeito de paralaxe 3D no logo da Hero Section
- Cursor personalizado com rastro suave
- Animações de entrada (fade-up) ao rolar
- Formulário Respondi integrado

## Como Rodar Localmente

Você pode abrir o arquivo `index.html` diretamente no navegador ou usar um servidor local:

```bash
npx serve .
```

## Personalização

- As cores principais estão definidas como variáveis no início do arquivo `styles.css` (`:root`).
- O texto e links podem ser alterados diretamente no `index.html`.
